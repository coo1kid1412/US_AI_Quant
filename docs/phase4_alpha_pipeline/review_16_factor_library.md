### 2.2 Alpha158 美股适配

- 股票池：S&P 500（~500只蓝筹股）
- 数据范围：2015-01-01 至 2020-08-01
- 因子数量：158 个内置因子（KBAR 9 + PRICE 4 + ROLLING 145）

- 标签：T+1 到 T+2 的日度收益率（Qlib默认）
- 已知问题：VWAP0 列有 >30% NaN（美股 VWAP 数据缺失较多）

### 2.4 因子评估工具

`FactorEvaluator` 类支持以下指标：
- **Pearson IC**：因子值与未来收益的截面相关系数
- **Rank IC**：因子排名与收益排名的 Spearman 相关系数
- **ICIR**：IC均值/IC标准差，衡量IC稳定性
- **因子换手率**：Top-20%持仓的日度变化比例
- **共线性检测**：因子间相关系数 > 0.7 的配对识别

性能优化：使用 `groupby.transform()` 向量化计算替代双重循环，158因子的 IC 计算时间从 10分钟+ 降至约1分钟。

## 4. 关键分析

### 4.1 为什么美股单因子 IC 接近 0？

1. **市场效率**：美股是全球最高效的市场，简单因子的 Alpha 被迅速套利
2. **因子拥挤**：Alpha158 的因子都是公开的经典因子，机构投资者早已在使用
3. **时变性**：因子效果随市场环境剧烈变化（牛市/熊市/震荡期因子方向不同）
4. **Alpha158 适配性**：该因子集为中国 A 股设计，A 股散户占比高、效率较低，因子更容易"赚钱"

### 4.2 这对后续工作的影响

| 影响 | 说明 |
|------|------|
| ML 模型至关重要 | LightGBM 需要学习因子的**非线性组合**和**时变关系** |
| 滚动训练必不可少 | 因子-收益关系在变化，模型必须定期重训 |
| RD-Agent 价值凸显 | 需要自动化挖掘**美股特有**的新因子 |
| 特征工程可改进 | 考虑加入宏观因子（利率、VIX）、行业因子、另类数据 |

### 4.3 共线性问题

617 对高共线性因子（如 SUMP/SUMN/SUMD 系列完全相关），建议：
- 在模型训练前进行因子去重（ACT_17 中处理）
- 每组共线性因子只保留 IC 最高的一个
- 或依赖 LightGBM 的内置特征选择能力
